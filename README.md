## 🚀 Overview

An intelligent, real-time **voice-based interview practice application** that simulates realistic job interviews across multiple languages.

This project helps users improve communication skills, confidence, and performance through **natural spoken interaction** with an AI interviewer.

## 🎯 Features

- 🎙️ Real-Time Voice Interaction  
- 🌍 Multilingual Interviews (English, German, Spanish…)  
- 🧠 AI-Powered Questioning  
- 🔄 Context-Aware Follow-Ups  
- 📊 Instant Spoken Feedback  
- 🎯 Adaptive Difficulty  
- 🗣️ Human-Like Conversation Flow  

---

## 🧩 How It Works

1. User speaks via microphone  
2. Speech → Text (STT)  
3. AI processes response  
4. AI generates next question  
5. Text → Speech (TTS)  
6. Conversation continues  

---

## 🏗️ Tech Stack

- **Frontend:** React / Next.js  
- **Backend:** Node.js / Python  
- **STT:** Whisper  
- **TTS:** OpenAI TTS / ElevenLabs  
- **AI:** GPT-based models  
- **Automation:** n8n (optional)

## ⚙️ Environment Variables

Create a `.env` file:

```env
API_KEY=your_api_key
STT_PROVIDER=whisper
TTS_PROVIDER=your_tts_service
DEFAULT_LANGUAGE=en
```
